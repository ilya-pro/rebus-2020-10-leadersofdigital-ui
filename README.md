## Project setup
Install Vue CLI Tool global if not installed
```
npm install -g @vue/cli
```
update dependency modules
```
npm install
```
Run Vue UI tool
```
vue ui
```
Plugin for Chrome
[Vue.js devtools](https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd)
